○ Descripción general del sitio web
Sitio web solicitado por la empresa OnlyFlans, especialistas en flanes, este sitio se realizó según las especificaciones, utilizando html con las potencialidades del framework de bootstrap y Django, el trabajo se desarrollo pensando en un sitio funcional con la promoción de la fidelización de los clientes y potenciales clientes.

○ Características base del sitio web
Cómo características bases del sitio tenemos las páginas solicitadas de inicio, acerca (que fue modificado a Nosotros), Contacto e Iniciar Sesión
Una vez que el usuario ingresa a su sesión, el botón cambia a cerrar sesión y se muestra contenido con flanes exclusivos para esos clientes.


○ Características personalizadas añadidas al sitio web
Cómo caracteristicas añadidas se realizaron algunas modificaciones de diseño, para mejorar la visualización, se puso un aviso de cierre de sesión exitoso y se agrego la opción de Hacer pedido, el cual está diferenciado para clientes no inscritos y clientes inscritos, a parte se invita a comprar e iniciar las sesiones
